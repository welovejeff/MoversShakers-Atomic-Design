Help me create a TypeScript function that validates email addresses using a regex pattern. Include proper error handling and JSDoc comments.

Here's a well-documented TypeScript function for email validation:

typescript 
/**
 * Validates an email address using RFC 5322 compliant regex
 * @param email - The email address to validate
 * @returns Object containing validity status and optional error message
 */
function validateEmail(email: string): { 
  isValid: boolean; 
  error?: string 
} {
  // Check for empty input
  if (!email || email.trim().length === 0) {
    return { isValid: false, error: 'Email address is required' };
  }

  // RFC 5322 compliant email regex
  const emailRegex = /^[a-zA-Z0-9.!#$%&'*+/=?^_`{|}~-]+@[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?(?:\.[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?)*$/;

  if (!emailRegex.test(email)) {
    return { isValid: false, error: 'Invalid email format' };
  }

  return { isValid: true };
}

This function includes:

  • RFC 5322 compliant regex pattern
  • Empty input validation
  • Descriptive error messages
  • Full JSDoc documentation
