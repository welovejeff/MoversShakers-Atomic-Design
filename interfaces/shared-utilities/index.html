Usage Guidelines

Keyboard First

All utilities support keyboard navigation. Users should be able to complete any action without a mouse.

Accessible

ARIA labels, focus management, and screen reader announcements are built into every component.

Performant

Optimized for 60fps animations. Virtualized lists for large datasets. Debounced search inputs.

Composable

Each utility is a standalone component that can be composed with others to build complex interactions.
