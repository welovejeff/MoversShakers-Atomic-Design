What is an Atomic Design System?

Atomic Design is a methodology for creating design systems. It breaks down interfaces into fundamental building blocks called atoms, which combine to form molecules, organisms, templates, and pages.

How do I use this design system?

Simply copy the HTML components you need and paste them into your project. Make sure to include the Tailwind CSS CDN and the custom CSS variables defined in the head section.

Can I customize the colors?

Yes! All colors are defined as CSS custom properties in the :root selector. Simply update the hex values to match your brand colors.
